# Hashi (橋) - Future Zen Portfolio

> A digital bridge between traditional aesthetics and future logic.

This portfolio represents my journey as a developer, blending the principles of "Ma" (negative space) with cutting-edge web technologies.

**Live Demo:** [https://hashi173.github.io/hashi-portfolio/](https://hashi173.github.io/hashi-portfolio/)

## Tech Stack

- **Framework:** React 19 (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Design:** "Future Zen" (Custom Calligraphy & Watercolor Aesthetics)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hashi173/hashi-portfolio.git
   cd hashi-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```

## Deployment

This project handles deployment to GitHub Pages automatically via `gh-pages`.

```bash
npm run deploy
```
